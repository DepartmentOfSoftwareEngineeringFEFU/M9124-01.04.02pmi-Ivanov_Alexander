# Использование методов компьютерного зрения для поиска и 3д реконструкции объектов подводной промышленной инфраструктуры

Программное средство для поиска объектов подводной промышленной инфраструктуры на подводных стерео снимках и их трехмерной реконструкции.

## Инструкция по использованию

Скомпилированный исполняемый файл: `x64/Release/CompVision.exe`

1. Запустите CompVision.exe.

2. Выберите папку с фоновыми изображениями (используются для построения модели фона и поиска объектов).  

3. Выберите папку со стерео снимками.

   Формат имён файлов стерео снимков:
   - Левый снимок: `frame_<номер>_0.bmp`
   - Правый снимок: `frame_<номер>_1.bmp`
   - `<номер>` – число, начиная с `00000` (например, `frame_00000_0.bmp`, `frame_00000_1.bmp`, `frame_00001_0.bmp`, …).

4. нажмите кнопку «Найти снимки с объектами».  
   Программа проанализирует стереопары, сравнивая их с фоновыми изображениями, и определит кадры, на которых присутствуют объекты, подлежащие реконструкции.  
   В результате будет создан файл `object_frames.txt` в папке со стереопарами. В этом файле перечислены номера снимков для дальнейшей реконструкции.

3. Выберите файл с номерами снимков `object_frames.txt`.

3. Выберите текстовый файл параметров стереокамеры следующего формата:
    Первая строка: расстояние между камерами (мм)
    Вторая строка: фокусное расстояние (мм)

5. Нажмите кнопку «Начать реконструкцию».  
   Программа выполнит вычисление модельных матриц, карт глубины и построение трёхмерной модели сцены.

   По окончании процесса откроется отдельное окно с визуализацией полученной 3D-модели.

6. Трёхмерная модель будет автоматически сохранена в файл `out.obj` в папке стерео снимков.

## Пример работы на наборе данных

Папка /test содержит набор данных для тестирования. Для использования укажите в программе:
- Папка с фоновыми изображениями: /test/bg
- Папка со стерео снимками: /test
- Файл параметров стереокамеры: /test/camera_params.txt

## Сборка проекта

Для сборки исходного кода требуется **Visual Studio 2022** с установленными компонентами для разработки на C++.

### Необходимые зависимости

| Библиотека        | Версия      | 
|-------------------|-------------|
| **Eigen**         | 3.4.0       |
| **OpenCV**        | 4.8.1       |
| **OpenCV_contrib**| 4.x         |
| **Open3D**        | 0.18.0      |
| **wxWidgets**     | 3.3.1       |

### Порядок сборки

1. Откройте файл решения (`.sln`) в Visual Studio 2022.
2. Убедитесь, что все пути к зависимостям корректно указаны в настройках проекта.
3. Выберите конфигурацию `Release` и платформу `x64`.
4. Выполните сборку (Build → Build Solution).

Исполняемый файл будет расположен в `x64/Release/CompVision.exe`.
